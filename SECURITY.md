# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| latest (`main`) | ✅ |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report them via email to **ryanjbush@gmail.com** with the subject line `[SECURITY] Nova AI — <brief description>`.

Include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any suggested remediation if known

You can expect an acknowledgement within **48 hours** and a resolution timeline within **7 days** for critical issues.

## Data & Privacy Note

This project uses **fully synthetic patient data** generated for demonstration purposes. No real patient data, PHI, or PII is included anywhere in this repository. If you believe any real health data has been inadvertently included, please notify immediately via the contact above.

## Scope

This project is a portfolio/demonstration platform and is not deployed in a clinical or production environment.
